# How to Run the Cinema Ticket System

This guide provides step-by-step instructions on how to run the Cinema Ticket System application on your local machine.

## Prerequisites

Before you begin, ensure you have the following installed:

*   [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0) or a later version.

## Running the Application

1.  **Open a terminal or command prompt.**

2.  **Navigate to the project's root directory:**
    ```bash
    cd /Users/macbookpro/Downloads/Project/Practice/CinemaTicketSystem/CinemaTicketSystem
    ```

3.  **Run the application using the `dotnet run` command:**
    ```bash
    dotnet run
    ```

4.  **Access the application in your web browser:**
    Once the application is running, you will see output in the terminal indicating that the web server has started. You can access the application by opening your web browser and navigating to one of the following URLs:

    *   `http://localhost:5000`
    *   `https://localhost:5001`

## Default Admin Credentials

The application is seeded with a default administrator account:

*   **Email:** `admin@cinema.com`
*   **Password:** `Admin@123`

You can use these credentials to log in and access the admin dashboard to manage movies and screenings.
